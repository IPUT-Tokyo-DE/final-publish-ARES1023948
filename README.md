# ここにゲームのタイトルを記述する
戦車隊　VS 怪獣
## ゲームの内容
４台の戦車を操作して街を守りながら怪獣を撃退する！

![画面イメージ](docs/images/game_image01.png)

## 操作説明

- WASDキーで移動して右クリックでバリア展開
- マウスホイールで戦車同士が集散する
